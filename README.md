# Danny's Portfolio

This is my portfolio website, showcasing my experience as a Product Manager.

## Tech Stack

- **Structure**: HTML5
- **Styling**: [Bootstrap 5](https://getbootstrap.com/), [Animate.css](https://animate.style/), [Bootstrap Icons](https://icons.getbootstrap.com/)
- **Logic**: Vanilla JavaScript
- **AI Chatbot**: Built with [n8n](https://n8n.io/) and hosted on Google Cloud Platform (GCP)
- **Deployment**: Served directly from GitHub Pages
